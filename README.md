# 3-Column Preview Card Component

A responsive 3-column preview card component built as part of a Frontend Mentor challenge.  
The project focuses on **semantic HTML, clean Flexbox layout logic, and maintainable CSS**, avoiding fixed heights and absolute positioning for structural layout.
